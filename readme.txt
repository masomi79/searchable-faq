== Description ==
Searchable FAQ is a WordPress plugin that generates an FAQ section and makes it easily searchable. With this plugin, users can search FAQs by keywords and quickly find the information they need.

== Installation ==
1. Download the plugin file.
2. In the WordPress admin panel, go to the "Plugins" menu and select "Add New".
3. Click the "Upload Plugin" button, select the downloaded file, and install it.
4. Activate the plugin.

== Frequently Asked Questions ==
= How do I configure the plugin? =
After activating the plugin, you can add FAQs and configure the search functionality from the settings page.

= Where can I get support? =
Support is available on the plugin's official page or the support forum.
